### 2. 图片压缩——三档预设 + 自定义参数

压缩模式分三档：

- **缩小**：尽可能减小文件体积，适合上传到有大小限制的平台
- **均衡**：在画质和体积之间取中间值
- **清晰**：优先保留画质，轻度压缩

如果三档都不符合你的需求，还能自己设定清晰度百分比、输出尺寸、目标文件大小，精确到具体数值。
